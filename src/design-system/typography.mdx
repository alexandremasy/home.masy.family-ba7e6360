## Scale

Eleven rungs, and no others: `styles.css` wipes Tailwind's scale first, so `text-[13px]` is
unwritable. The names are Tailwind's, the values are Figma's — `text-xl` is **24px**, not 20.

<Typeset
  fontSizes={[10, 12, 14, 16, 20, 24, 28, 32, 40, 48, 56]}
  sampleText="Reads at arm's length"
  fontWeight={400}
/>

| rung        | px  | role                        |
| ----------- | --- | --------------------------- |
| `text-2xs`  | 10  | uppercase labels            |
| `text-xs`   | 12  | metadata, figures, captions |
| `text-sm`   | 14  | body, the default           |
| `text-base` | 16  | body, roomy                 |
| `text-lg`   | 20  | the name of a thing         |
| `text-xl`   | 24  | section title               |
| `text-2xl`  | 28  |                             |
| `text-3xl`  | 32  | page title                  |
| `text-4xl`  | 40  | the big number              |
| `text-5xl`  | 48  |                             |
| `text-6xl`  | 56  | display ceiling             |

## Tracking

`tracking-eyebrow` — 0.18em, for uppercase text. `h1`–`h3` get `-0.01em` from the stylesheet.
