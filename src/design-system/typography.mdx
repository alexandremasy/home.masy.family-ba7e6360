The names are Tailwind's, the values are Figma's, and they disagree by one rung: `text-xl` is
**24px** here, not Tailwind's 20. Read the value, not the name.

## One family

There is **one** typeface: the native OS stack. No webfont is loaded, so the face is San Francisco
on Apple, Segoe UI on Windows, Roboto on Android.

`--font-sans` and `--font-serif` currently resolve to the **same stack** — `.font-serif` is a
_display role_, not a second family, and it buys nothing today beyond marking intent at ~145 call
sites. Either it earns a real display face, or it should be replaced by the weight and tracking it
actually stands for. Until that is decided, the two samples below are identical on purpose.

## Eyebrow tracking

The one letter-spacing this system owns: `--tracking-eyebrow`, **0.18em**. Figma sets
letter-spacing 0 on every token, but the eyebrow is a dashboard pattern the reference doesn't
cover — so the app owns its spacing. One value, where there used to be seven. It belongs to
`components/eyebrow.tsx`; hand-spacing an uppercase label means you skipped the component.
