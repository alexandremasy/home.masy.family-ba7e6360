## The scale

Tailwind's default scale, base **4px** — `1` is 4px, so the number is the value divided by four.
The product lives almost entirely in the 2 → 6 range; anything above 8 is layout, not spacing.

## The rhythms that are decided

These are not free choices — they are the ones the components hold, so a page composes rather
than re-invents.

| Rhythm            | Value                                       | Where                                                   |
| ----------------- | ------------------------------------------- | ------------------------------------------------------- |
| Card padding `md` | `px-5` → `px-6` at `sm`                     | the default box, what 25 hand-built panels converged on |
| Card padding `sm` | `px-4`                                      | dense boxes, bento pills                                |
| Card body         | `py-4`                                      | between header and footer                               |
| Icon ↔ label      | `gap-2.5`                                   | every card header, every row                            |
| Sibling gap       | `gap-3`                                     | lists, chip rows                                        |
| Grid gap          | `gap-3` → `gap-4` at `sm` → `gap-5` at `lg` | the bento, below                                        |
| Section gap       | `space-y-8`                                 | between blocks of a page                                |

## `--nav-h` — the sticky bar

Currently <TokenValue token="nav-h" fallback="56px" />. Anything that sticks in a full-bleed page
must offset by this height, or it hides behind the bar. It mirrors the header in `_app.tsx` (h-14).

## `.grid-bento` — the dashboard mosaic

2 columns on mobile, 4 at ≥640px, 6 at ≥1024px, with fixed row heights on desktop
(`11rem`, then `12rem`) and `grid-auto-flow: row dense` so a small tile backfills a hole. Resize
the preview to watch it reflow.

Placement is never a card's business — `BentoItem` carries it. See **Layout → Bento Grid**.
